Welcome to the Terri Quintel Astrology assessment! This project contains two frontend tasks that should be completed within 1-2 days.

## Quick Start

1. **Start the Backend:**
   ```bash
   cd backend
   npm install
   npm start
   ```
   The API will run on `http://localhost:3000`

2. **Start the Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
   The app will run on `http://localhost:4200`

3. **Navigate to Tasks:**
   - Task 1: http://localhost:4200/task1
   - Task 2: http://localhost:4200/task2

## Task Overview

### Task 1: Display Astrological Charts
Create a component that fetches and displays astrological charts from the API. See `frontend/src/app/task1/task1.component.ts` for detailed requirements.

### Task 2: Birth Chart Calculator
Create a form component that allows users to calculate their birth chart. See `frontend/src/app/task2/task2.component.ts` for detailed requirements.
